# Block-Ads-by-UserScripts

## 소개
이 저장소에는 애드가드에서 차단되지 않는 웹사이트의 광고나 배너를 차단하기 위한 사용자 스크립트가 포함되어 있습니다.

## 스크립트 목록

### 1. 뉴덕 광고 차단 스크립트
- 파일: `newduck_ad_blocker.user.js`
- 설명: newduck.net 사이트의 모든 광고 영역을 차단합니다 (게시판 및 게시글 페이지)

## 사용 방법

### 1. Userscripts 앱 설치 및 설정 (iOS/iPadOS)
1. App Store에서 "Userscripts" 앱을 다운로드합니다.
2. 설치 후 사파리 설정에서 Userscripts 확장 프로그램을 활성화합니다:
   - 사파리 > 설정 > 확장 프로그램 > Userscripts > 사용 활성화

### 2. 스크립트 설치
1. Userscripts 앱에서 스크립트 추가를 선택합니다.
2. 이 저장소의 `.user.js` 파일을 복사하여 새 스크립트로 붙여넣기합니다.
   또는 각 스크립트 파일의 "Raw" 버튼을 클릭하여 직접 설치할 수 있습니다.
3. 스크립트 이름과 설명을 적절히 입력합니다.
4. 스크립트를 저장합니다.

### 3. 스크립트 사용
1. 스크립트가 적용될 웹사이트에 접속합니다.
2. 사파리 주소창 옆의 Userscripts 아이콘을 클릭합니다.
3. 해당 웹사이트에 적용할 스크립트를 활성화합니다.

## 스크립트 커스터마이징
각 스크립트 파일의 상단 주석에서 다음 부분을 수정하여 적용할 웹사이트를 변경할 수 있습니다:
```javascript
// @match        https://example.com/*
```

여기서 `https://example.com/*`를 원하는 웹사이트 URL 패턴으로 바꿉니다.

## 문제 해결
- 스크립트가 작동하지 않는 경우, 사파리 개발자 도구를 사용하여 콘솔에 오류가 있는지 확인합니다.
- 웹사이트의 구조가 변경되면 스크립트를 업데이트해야 할 수 있습니다.

## 맞춤형 스크립트 요청
특정 웹사이트에 대한 맞춤형 스크립트가 필요한 경우:
1. 차단하려는 요소가 있는 웹사이트 URL을 제공합니다.
2. 차단하려는 요소의 스크린샷이나 자세한 설명을 제공합니다.
